# Roadmap

Shipped functionality is described by the concept docs in this bundle and by the
versions published to the [Bazel Central Registry](https://registry.bazel.build/modules/gavel_tools) — this
file carries **only what is not yet built**, so it does not drift against the
code. An entry lives here until it ships, then it leaves.
